how to use
Place this file at editor/editor.html in your Jekyll repo, then click connect repo and pick the repo root (the folder containing _posts/). The browser remembers it.

save to _posts/ writes the post directly. Image uploads go straight into assets/images/. Folders are created if missing.

Works in Chrome, Edge, Brave, Arc. Firefox/Safari fall back to download.
